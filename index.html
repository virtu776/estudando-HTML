Resumo sobre HTML5

  • Paragrafos <p>

    • Você pode escrever os paragrafos de qualquer jeito basta colocar tudo no meio do par de tag <p> e >/p>

    *Para pular paragrafos no texto, é preciso usar a tag <br>.

    ex: -ola mundo!
    estou programando em CSS,
    mas não vejo a hora de programar em JAVA.

  • Simbolo Especias

    • ® = marcas registrada
    © = copyright
    ↑ = seta para cima
    ↓ = seta para baixo
    ™ = MT
    -Moedas:
    € = euro
    £ = pound
    ¥ = yen

  • Emojis

    • No site Emojipidia.com, é possivel encontrar os variados tiposde emojis e seus "codepoints", que são codigo dos emojis para adicionar no script do site, pois não colar o emoji na script.
    Para adicionar um emoji no site, se utiliza o comando "&#x" e logo em seguida o codepoints.
    ex: 😏

    kjkvjlbl
